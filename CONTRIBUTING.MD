## Getting Started

1. Fork the repository.
2. Clone your fork locally.
3. Create a new branch for your changes.
4. Make your changes and test them.
5. Commit and push your changes.
6. Open a Pull Request.

---

## Prerequisites

Before contributing, ensure you have:

* Git installed
* Node.js and npm installed
* A GitHub account
* Basic knowledge of React and JavaScript

---

## Setting Up the Project

### Clone the Repository

```bash
git clone https://github.com/YOUR-USERNAME/LegalEase.git
cd LegalEase
```

### Install Dependencies

```bash
npm install
```

### Run the Development Server

```bash
npm run dev
```

---

## Contribution Workflow

### Create a Branch

```bash
git checkout -b feature/your-feature-name
```

### Make Changes

Implement your feature, bug fix, or documentation improvement.

### Commit Changes

```bash
git add .
git commit -m "docs: add contribution guidelines"
```

### Push Changes

```bash
git push origin feature/your-feature-name
```

---

## Commit Message Guidelines

Use clear and descriptive commit messages.

Examples:

```text
feat: add user authentication
fix: resolve navbar responsiveness issue
docs: improve README documentation
refactor: optimize API handling
```

---

## Pull Request Process

1. Ensure your branch is up to date.
2. Verify that your changes work correctly.
3. Create a Pull Request with a clear title and description.
4. Link related issues if applicable.
5. Wait for maintainer review and feedback.

---

## Reporting Issues

When reporting a bug, include:

* Issue description
* Steps to reproduce
* Expected behavior
* Actual behavior
* Screenshots (if applicable)
* Environment details

---

## Suggesting Features

When proposing a feature:

* Clearly explain the feature.
* Describe the problem it solves.
* Provide possible implementation ideas.
* Mention expected benefits.

---

## Code of Conduct

By participating in this project, you agree to follow the guidelines outlined in the project's CODE_OF_CONDUCT.md.

Let's work together to build a respectful, collaborative, and inclusive community.
